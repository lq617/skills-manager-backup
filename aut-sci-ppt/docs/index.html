Skill Modules

六大技能详情

每个技能独立可用，也可组合成完整科研工作流

🔍 sci-search
文献检索与期刊指标
支持检索 arXiv、PubMed 与 Web of Science 三大数据源，自动附加 JCR 分区、影响因子，高亮 Nature / Science 等顶刊结果。
搜索文献 查文献 影响因子 find papers
📄 sci-extract
论文核心内容提取
采用Heilmeier范式，从 PDF 中深度提取研究问题、方法论、关键结论、实验参数与数值对比，超越简单摘要，输出结构化结果。
分析论文 提取结论 论文要点 analyze paper
🖼️ sci-figure
论文图表智能提取
自动检测 PDF 中的图表位置，高精度裁剪，支持复合图拆分为独立子图，保留出版级分辨率（600 DPI）。
提取图片 截取子图 extract figure
✍️ sci-review
综述写作与审稿回复
基于 4 阶段系统综述结构辅助撰写文献综述；专业审稿回复模板，对标 NeurIPS / ICLR 标准，逐条精准回应。
文献综述 审稿回复 论文润色 rebuttal
📚 sci-zotero
Zotero 文献库集成
同步 Zotero 远程文献库，支持通过 DOI / ISBN / PMID 添加引用，自动获取开放获取 PDF，无缝融入写作流程。
同步文献 Zotero add citation
📊 sci-ppt
学术 PPT 自动生成
从论文 PDF 或结构化文本一键生成专业学术 PPT，支持 LaTeX 公式渲染、图片嵌入、中英双语布局，适配组会与答辩场景。
生成PPT 组会汇报 答辩PPT 论文转PPT
Quick Start

三步完成安装

一条命令注册全部技能，立即开始使用

1
安装全部技能
bash 
npx skills add ShZhao27208/Aut_Sci_Write -g -y
2
配置可选环境变量
bash 
# Zotero 集成（可选）
export ZOTERO_API_KEY=your_api_key
export ZOTERO_USER_ID=your_user_id

# Web of Science API（可选）
export WOS_API_KEY=your_wos_key
3
在 Claude Code 中触发技能
claude code 
# 直接用自然语言触发，例如：
搜索文献 transformer attention mechanism

# 或使用斜杠命令：
/sci-search "large language models 2024"
